# Homepage video slots

The premium homepage is wired for these original, licensed MP4 assets:

- `racing-hero.mp4`
- `parade-ring.mp4`
- `track-motion.mp4`

The hero uses `preload="none"`, autoplay, muted playback, looping, and
`playsInline`. If none of these files are present or a browser cannot play
them, the dark racing-green motion treatment in `app/globals.css` remains
visible as the fallback.

Recommended delivery settings:

- H.264 MP4
- 1920×1080 or 1600×900
- 8–15 second seamless loop
- No embedded text, logo, audio, or recognizable third-party branding
- Aim for less than 4 MB per loop for mobile performance
